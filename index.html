Projects

Vehicle Parking System

Vehicle Parking System is a technology-driven solution designed to manage and optimize the allocation of parking spaces in a given area. It allows users to easily find available parking spots, track the occupancy status in real time, and efficiently park their vehicles. The system can be integrated with databases to store vehicle and parking information, enabling features like parking slot reservation, time-based pricing, and automated billing. Such systems aim to reduce congestion, improve parking efficiency, and enhance the overall user experience.

Text-Based Adventure Game

Text-based adventure game, developed by Jerson A. Mirafuentes and Niel Angelo L. Remotigue, immerses players in a dynamic and suspenseful narrative set in the mysterious Dark Lands. The game challenges players to make critical decisions as they navigate through dangerous paths, including eerie forests, rushing rivers, and abandoned huts, all while managing their health and strategic use of skills like Fireball, Defensive Shield, and Dash Movement. With multiple choices and outcomes, players must think carefully to survive and ultimately find safety. This project highlights your creativity, storytelling, and programming skills, offering an engaging and interactive gaming experience.

Python Calculator

Python Calculator project is a user-friendly application designed to perform basic arithmetic operations such as addition, subtraction, multiplication, and division. Built using Python's simplicity and versatility, the calculator provides a clean interface for input and output, ensuring ease of use for users of all technical backgrounds. With efficient logic handling and robust error-checking for invalid inputs, the program ensures reliability and accuracy. This project demonstrates a practical application of Python programming and showcases your problem-solving skills and understanding of fundamental programming concepts.
